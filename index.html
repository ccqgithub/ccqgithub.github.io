WAI-ARIA 无障碍网页应用技术

By season.chen

发表于 2015-03-18

WAI-ARIA[1] , 是Web Accessibility Initiative - Accessible Rich Internet Applications 的缩写，指无障碍网页应用技术。她主要解决的一个问题：让残障人士能无障碍地访问网页上的动态内容。

the Accessible Rich Internet Applications Suite, 可帮助访问Web内容和Web应用有困难的用户进行访问的方法（比如用静态的(screen-reading technologies)浏览器浏览用Ajax技术制作的动态网页）。它尤其帮助动态内容和用Ajax, HTML, JavaScript和相关技术开发的高级用户接口控件。

从今天起，沉下心来，记点东西！

By season.chen

发表于 2015-03-18

很早以前就准备自己建个博客，从新浪博客到wordpress，再从wordpress到hexo，再从hexo到自己建站，然后回归hexo。

瞎折腾了许久，可转首一看，却发现什么也没留下。而反观别人的博客，却像森林一般生机勃勃，越来越茂盛。

这不禁让我开始反思，我这些时间都做了些什么？

一路走来，学了不少东西，可也发现自己在渐渐遗忘。人的记忆不是恒久的，有些东西必须要记录下来才有意义。

回想当初，也许是有一点强迫症作怪，做了又改，改了又做，总觉得不够尽善尽美。如此往往复复也不知经历了多少个版本，时间溜走了，东西却没留下。

直至如今，才发觉自己一直走在一条错误的道路上。

经历了太多才明白，其实这世上根本就没有完美的东西。

大道至简，空有其表没有内容的东西，终究没有什么意义。

相反，简简单单的东西，积累起来，那才是一笔宝贵的财富。

所以，从今天去，沉下心来，认认真真记录一点东西。

幸好，有些牛人，早已替我铺好了路。

感谢 hexo ，让我轻而易举地能迈开脚步。

也感谢hexo主题 jacman，让我的博客能轻而易举地看起来漂亮。
