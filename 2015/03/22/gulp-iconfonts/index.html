使用gulp很容易地创建本地iconfont（矢量图标）字体库

By season.chen

发表于 2015-03-22

文章目录

iconfont越来越流行，出现了像阿里巴巴矢量图标库，IcoMoon等优秀的在线字体管理工具。

但是如果我们需要在没有网的时候使用本地图标库的时候怎么办？通过gulp能很容易的实现。我的一个例子。

使用很简单，克隆代码.

// 切换目录
cd dir
// 安装依赖包
npm install
// 生成html、css、fonts
gulp

然后打开‘out/index.html’,你就会发现svgs目录下面的所有svg图标都转为iconfonts了，是不是soeasy！！！

当然你还可以自己修改‘tpl/*’里面的模板文件和‘gulpfile.js’来生成你自己的格式哦！！！
