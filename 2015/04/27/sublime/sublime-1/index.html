Sublime Text 是一个代码编辑器（Sublime Text 2是收费软件，但可以无限期试用），也是HTML和散文先进的文本编辑器。Sublime Text是由程序员Jon Skinner于2008年1月份所开发出来，它最初被设计为一个具有丰富扩展功能的Vim。

安装

  • Sublime Text 安装

Package Control

Package Control 是Sublime Text的包管理器。里面是Sublime Text的各种插件、主题……

  • 安装Package Control

原来在Sublime text3上安装Package Control很麻烦,现在可以有简单的方法了。
使用Ctrl+`快捷键或者通过View->Show Console菜单打开命令行,粘贴如下代码:

import urllib.request,os; pf = 'PackageControl.sublime-package'; ipp = sublime.installed_packages_path();urllib.request.install_opener( urllib.request.build_opener(urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf),'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace('','%20')).read())
  • 安装插件
    按快捷键ctrl+shilft+p打开命令面板，输入package选择pacakge install,然后跳出包列表，输入关键字选择需要的包即可

配置

{
    "color_scheme": "Packages/User/Flatland Dark (SL).tmTheme",
    "font_size": 14,
    "highlight_line": true,
    "ignored_packages":
    [
        "Theme - Phoenix",
        "Dayle Rees Color Schemes",
        "SublimeLinter",
        "EsFormatter"
    ],
    "phoenix_color_blue": true,
    "theme": "Flatland Dark.sublime-theme",
    "translate_tabs_to_spaces": false,
    "word_wrap": true
}
